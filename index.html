Recent Projects

TCP Chat

A TCP chat app allows users to communicate over a network using TCP. It's reliable, connection-oriented & sends messages in real-time. Messages are divided into packets & reassembled. It's used in online gaming, customer service, and remote collaboration for fast & reliable communication.

Java Bank

This project, Java-Bank, was developed as part of the Academia De Código bootcamp. Java-Bank focuses on teaching core Java concepts and introducing libraries, frameworks, and tools that are commonly used in building modern applications. The project is centered around a bank theme, which encompasses a range of functionalities and responsibilities typical of enterprise applications. These include building user interfaces, implementing client/server architecture, working with relational databases, and managing database transactions.

MapEditor

Map Editor is a user-friendly paint application that has been designed specifically for creating and editing maps. With its grid-based interface, the application allows you to paint individual squares, erase previous paint, clear the entire board, save your work for later use, and load previous maps. The minimalistic design of Map Editor makes it incredibly easy to use, even for those who are not experienced with similar applications.

Snake Game

During the games week at Academia de Código, a group project was undertaken with the aim of recreating the classic game "Snake" using Java programming language. The game features a snake that moves around the screen, consuming food while avoiding its own tail. As the snake consumes food, it grows longer, making the game increasingly challenging. The objective of the game is to eat as much food as possible and achieve the highest score. Overall, the project was a fun and engaging exercise in game development.
